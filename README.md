# Big Data & Visual Analytics: Customer Satisfaction Dashboard

This project demonstrates a complete data lifecycle—from processing raw, semi-structured data to delivering actionable business insights.

## 🚀 Project Overview
* **Data Engineering:** Used **Apache Spark** to clean, transform, and aggregate a large-scale dataset (Yelp reviews), handling challenges like missing values and inconsistent categories.
* **Visual Analytics:** Developed an interactive **Tableau** dashboard to visualize customer engagement, sentiment trends, and geographical satisfaction.
* **Key Insights:** Correlated business categories with star ratings and identified the impact of global events (COVID-19) on customer activity.

## 🛠️ Tech Stack
* **Processing:** Apache Spark (PySpark)
* **Visualization:** Tableau Desktop / Tableau Prep
* **Storage:** CSV / Parquet
* **Analysis:** Exploratory Data Analysis (EDA), Sentiment Analysis, KPI Reporting

## 📂 Repository Structure
* `/sparkscripts`: Contains Python scripts used for distributed data processing.
* `/screenshots`: Includes high-resolution captures of the final Tableau dashboards and KPIs.

## 📊 Key Performance Indicators (KPIs)
- Total Customer Reviews
- Overall Average Rating
- Percentage of Positive vs. Negative Sentiment
